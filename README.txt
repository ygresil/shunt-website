Shunt-website
=============

Sources of Shunt’s website, made using Bootstrap 3 framework.

This website describe the application and all its features. It also contain the documentation for installing the application’s daemon on your desktop computer.

Shunt is a new mobile app (coming soon on android & iOS) which allow you to manage your downloads whenever you are. 

Made by the Shunt-team for ETNA’s EIP project :)
