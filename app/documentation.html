Documentation

Shunt peut être installé sur les systèmes Linux, Mac, et Windows. Les fichiers d'installation seront disponibles prochainement sur le site. Pour le moment, il vous faudra cloner le dépôt GIT pour les récupérer.

windows

Installation

Pour installer Shunt sous Windows, récupérez l'installeur disponible dans le sous-dossier dist/windows/x64 ou dist/windows/x32 dépendamment du système sur lequel vous êtes (64 ou 32 bits respectivement).

Lancez l'installateur en tant qu'administrateur et suivez les étapes d'installation.

Utilisation

Utilisez simplement les raccourcis créés sur le bureau pour lancer et arrêter Shunt. Vous pourrez alors vous connecter sur votre ordinateur devenu serveur Shunt via l'application mobile Shunt.

mac

Installation

Vous aurez besoin pour cette installation de récupérer les sources du projets grâce à GIT. Si il n'est pas encore installé sur votre ordinateur, téléchargez l'installeur via le lien sourceforge.net/projects/git-osx-installer et suivez les étapes d'installation.

Si NodeJS n'est pas encore installé sur votre ordinateur, téléchargez simplement la version 0.10.29 de NodeJS à partir du site officiel nodejs.org et installez-la.

Pour l'installation de Shunt à proprement parler, ouvrez un terminal, rendez-vous dans le dossier où vous souhaitez installer Shunt et récupérez les sources grâce à la commande :

git clone https://github.com/etna-bertra-n/Daemon

Rendez-vous ensuite dans le dossier que vous venez de cloner et installez Shunt :

cd Daemon && npm install

Utilisation

Pour lancer Shunt, ouvrez un terminal, rendez-vous dans le dossier d'installation de Shunt et lancez la commande suivante :

node .

Le Daemon démarrera. Pour l'arrêter, retournez dans la fenêtre de terminal sur laquelle vous l'avez lancé et faite CTRL+C (^C).

Linux

Installation

L'installation de Shunt a besoin de NodeJS pour fonctionner. Si NodeJS est déjà installé sur votre machine, installez seulement le paquet shunt.deb. Si vous avez besoin d'installer NodeJS, vous pouvez le faire seul ou à l'aide du script nodejs_install.sh que vous pourrez trouver dans le sous-dossier dist/linux du répertoire GIT cloné. Lancez alors le script et suivez les étapes d'installation.

Une fois NodeJS installé, vous pouvez passer à l'installation de Shunt grâce au paquet shunt.deb disponible également dans le sous-dossier dist/linux du répertoire GIT cloné. Pour installer Shunt à partir du paquet, lancez simplement la commande suivante :

#> dpkg -i shunt.deb

Le paquet a été testé et fonctionne sur un système Linux Debian Wheezy.

Utilisation

Pour lancer Shunt, utilisez le fichier init installé sur votre machine. Les commandes disponibles sont les suivantes :

service shunt-daemon [start|stop|restart]

Rendez-vous sur github.com/etna-dela-r/Daemon pour une utilisation avancée.
